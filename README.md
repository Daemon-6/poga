Lietotāja ceļvedis
    1. Spēles sākšana:
        ◦ Nospiežot pogu "Sākt", sāksies spēle.
    2. Pogas spiešana:
        ◦ Spiežot pogu, tiek skaitīts, cik reizes poga tika nospieta.
    3. Spēles beigas:
        ◦ Spēli beidz aizverot spēles logu.
        ◦ Pēc spēles beigām tiek parādīts, cik reizes poga tika nospieta un cik ilgs bija spēles laiks.
